# ますぐらぼ

数式初学者のための数学ツール。

## 概要

ますぐらぼ は、Scratchのようなブロック操作で数式を作り、
リアルタイムにグラフや計算結果を確認できる実験的なWebアプリです。

従来の数式入力では必要だった

* 括弧のミス
* 記号入力
* 数式構文の暗記

などを減らし、
「数学を構造として触る」体験を目指しています。

## コンセプト

### 数式を“文字列”ではなく“構造”として扱う

多くのグラフツールでは、数式をキーボードで入力します。

しかし、ますぐらぼでは

* 数字
* 演算子
* 関数
* 変数

をブロックとして組み合わせることで、
視覚的に数式を構築できます。

## 特徴

* ブロック式数式エディタ
* リアルタイムグラフ描画
* ドラッグ＆ドロップ操作
* Scratch風カテゴリーUI
* 関数アート制作支援

## 名前の由来
### 「math」「すぐ」「グラフ」「ラボ」
の4つを掛け合わせた、「数学のグラフをすぐに描けるラボ」にしたいという思いを込めています。

## 使用技術

現在使用している技術：

* HTML / CSS / JavaScript
* Blockly
* Desmos API

## 開発目的

数学表現では、
「数式を書けること」が創作の入口になっています。

ますぐらぼは、
数式の文法より先に、

* 試す
* 触る
* 組み替える
* 見て理解する

ことを重視したツールを目指しています。
